# Password Strength Analyzer

## Description
A Python-based Password Strength Analyzer that evaluates passwords based on:

- Length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

## Features
- Password strength checking
- Strong password suggestions
- Password reuse detection using SQLite database

## Technologies Used
- Python
- SQLite
- Regular Expressions

## How to Run

```bash
python password_analyzer.py
